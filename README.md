# MONSTER HUNTER Mini
MONSTER HUNTER Mini is a strategic single player experience inspired by the [Monster Hunter](https://en.wikipedia.org/wiki/Monster_Hunter) franchise.
You take the role of a Monster Hunter, seeking out monsters to defeat for glory, as well as various weapons and gear.
You'll get to select a hunt (with varying difficulties) to fight a monster; the more difficult the hunt the better the gear!
## FEATURES
- Different Difficulties (Low Rank, Medium Rank, High Rank, Very High Rank, Master Rank) 
- Risk/Reward System
- Hunter Inventory
## HOW TO PLAY
Choose from the five difficulties and battle it out! Keep in mind the higher the rank, the more rewards you get! Be wary though, you may lose
some gear if you fail to defeat the monster.
## INSTALLATION
1) Clone or Download the repository
2) Download [.NET](https://dotnet.microsoft.com/en-us/download/dotnet) + [Visual Studio 2026](https://visualstudio.microsoft.com)
3) Open in Visual Studio and run
## FRAMEWORK
- ASP.NET Core MVC with C#
- Entity Framework Core
- Bootstrap 5
- HTML/CSS/JavaScript
